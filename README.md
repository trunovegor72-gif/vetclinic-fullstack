# Информационная система ветеринарной клиники

Курсовая работа. Веб-приложение для управления ветеринарной клиникой:
онлайн-запись на приём, электронные медкарты питомцев, учёт вакцинаций,
интеграция с лабораторией.

## Стек

React (Vite) + Node.js (Express) + Apache Kafka + Redis + SQLite, всё в Docker.

## Запуск

```bash
docker compose up --build
```

После старта открыть `http://localhost:8080`.

## Роли

- **Владелец** — запись на приём, регистрация питомцев, медкарты.
- **Ветеринар** — приём, ведение медкарт, назначение анализов.
- **Администратор** — управление врачами, питомцами, записями, медкартами; мониторинг Kafka и Redis.

## Структура

```
vetclinic-fullstack/
├── docker-compose.yml
├── backend/    Node.js + Express + kafkajs + ioredis + SQLite
└── frontend/   React (Vite), за nginx
```
